# Markdown 基础语法指南
```
用于层级结构，使用 # 表示，最多 6 级标题
```
## 1. 标题
# 一级标题
## 二级标题
### 三级标题


## 2. 段落和换行
```
- 普通段落
强制换行，行尾加两个空格。或者使用html的<br>标签
```
- 普通段落：直接输入文字，空行分隔段落
- 换行：行尾加两个空格 或者 使用 HTML的br标签


## 3. 文字样式
```
**粗体文字**
*斜体文字*
***加粗+斜体***
~~删除线文字~~
`行内代码`
```
**粗体文字**  
*斜体文字*  
***加粗+斜体***<br>
~~删除线文字~~  
`行内代码`


## 4. 列表
### 无序列表
```
使用 - 或 + 或 * 开头
```
- 项目1
* 项目2
+ 项目3
  - 子项目3.1
  - 子项目3.2

### 有序列表
```
使用数字加点 1. 开头
```
1. 第一项
2. 第二项
3. 第三项

### 任务列表
```
- [x]
- [ ]
```
- [x] 已完成任务  
- [ ] 未完成任务
